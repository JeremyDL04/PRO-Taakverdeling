 
 0. Pak het zip-bestand eerst uit! Plaats de bestanden in je xampp/htdocs-map.

 1. Zorg ervoor dat je mappen-structuur klopt. Het moet er (ongeveer) zo uitzien;

 	C:/
 		xampp/
 			htdocs/
 				A2_startpunt/				<--- deze map kun je hernoemen
 					backend/
 					css/
 					img/
 					README_leesmij.txt
 					contact.php
 					footer.php
 					header.php
 					index.php


2. Let op: vanwege de php-code kun je dit project ALLEEN maar openen via xampp, zoals je geleerd hebt.

3. In verschillende bestanden zie je aanduidingen als:
	
	"START eigen code"
	"EINDE eigen code"

	Daartussen kun je dan zelf dingen wijzigen en toevoegen


4. Je komt regelmatig code tegen die je misschien nog niet kent; in de komende weken ga je bij WEB
   meer leren over hoe dit startpunt is opgebouwd. Voor nu mag je denken:

		_het werkt dus ik blijf er maar af_


5. Veel succes!
